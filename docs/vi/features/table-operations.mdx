# Thao tác với Bảng

Chuột phải bảng trong sidebar để drop, truncate hoặc nhân bản. Tạo và quản lý view. Chuyển đổi giữa các database trên cùng kết nối.

## Các Thao Tác với Bảng

Chuột phải bảng trong sidebar để truy cập các thao tác.

### Xóa Bảng (Drop Table)

Xóa vĩnh viễn bảng và toàn bộ dữ liệu.

1. Chuột phải bảng (hoặc chọn nhiều bảng)
2. Chọn **Delete**
3. Cấu hình tùy chọn trong hộp thoại xác nhận
4. Click **OK** để thực thi

<Warning>
Xóa bảng không thể hoàn tác. Luôn backup dữ liệu quan trọng trước khi xóa.
</Warning>

### Truncate Bảng

Xóa toàn bộ hàng, giữ nguyên cấu trúc bảng.

1. Chuột phải bảng (hoặc chọn nhiều bảng)
2. Chọn **Truncate**
3. Cấu hình tùy chọn trong hộp thoại
4. Click **OK** để thực thi

**Tùy chọn theo database:**

<Tabs>
  <Tab title="MySQL/MariaDB">
    | Tùy chọn | Mô tả |
    |----------|-------|
    | **Ignore foreign key checks** | Tạm tắt kiểm tra foreign key |
    | **Cascade** | Không hỗ trợ cho TRUNCATE |
  </Tab>
  <Tab title="PostgreSQL">
    | Tùy chọn | Mô tả |
    |----------|-------|
    | **Ignore foreign key checks** | Không hỗ trợ (dùng CASCADE) |
    | **Cascade** | Truncate tất cả bảng liên kết qua foreign key |
  </Tab>
  <Tab title="SQLite">
    | Tùy chọn | Mô tả |
    |----------|-------|
    | **Ignore foreign key checks** | Tạm tắt kiểm tra foreign key |
    | **Cascade** | Không hỗ trợ |
  </Tab>
</Tabs>

<Note>
Truncate nhanh hơn `DELETE FROM table` vì bỏ qua log xóa từng hàng. Đồng thời reset auto-increment counter.
</Note>

### Thao Tác Hàng Loạt

Chọn nhiều bảng và Drop hoặc Truncate tất cả cùng lúc. Tùy chọn áp dụng cho mọi bảng được chọn.

1. Giữ `Cmd` và click chọn nhiều bảng
2. Chuột phải chọn **Delete** hoặc **Truncate**
3. Cấu hình tùy chọn một lần
4. Tất cả bảng được xử lý

## Quản Lý View

View là bảng ảo định nghĩa bằng SQL. TablePro hiển thị view trong sidebar với biểu tượng mắt và màu tím.

### Tạo View

Tạo view bằng cách mở tab SQL editor với template:

1. Chuột phải sidebar và chọn **Create New View...**
2. Hoặc **File** > **New View...**
3. Tab editor mới mở với template CREATE VIEW
4. Sửa tên view và query SELECT
5. Thực thi để tạo view

Template tự điều chỉnh theo database:

<Tabs>
  <Tab title="MySQL/MariaDB">
    ```sql
    CREATE VIEW view_name AS
    SELECT column1, column2
    FROM table_name
    WHERE condition;
    ```
  </Tab>
  <Tab title="PostgreSQL">
    ```sql
    CREATE OR REPLACE VIEW view_name AS
    SELECT column1, column2
    FROM table_name
    WHERE condition;
    ```
  </Tab>
  <Tab title="SQLite">
    ```sql
    CREATE VIEW IF NOT EXISTS view_name AS
    SELECT column1, column2
    FROM table_name
    WHERE condition;
    ```
  </Tab>
</Tabs>

### Chỉnh Sửa Định Nghĩa View

Sửa SQL của view hiện có:

1. Chuột phải view trong sidebar
2. Chọn **Edit View Definition**
3. Định nghĩa hiện tại mở trong tab editor mới
4. Sửa query và thực thi để cập nhật

<Note>
MySQL/MariaDB dùng `ALTER VIEW`. PostgreSQL dùng `CREATE OR REPLACE VIEW`. SQLite không hỗ trợ ALTER VIEW, phải drop và tạo lại.
</Note>

### Xóa View

Xóa view khỏi database:

1. Chuột phải view trong sidebar
2. Chọn **Drop View**
3. Xác nhận xóa

<Tip>
View không lưu dữ liệu. Xóa view chỉ xóa định nghĩa. Dữ liệu bảng gốc không bị ảnh hưởng.
</Tip>

## Thao Tác với Database

### Tạo Database

Tạo database mới từ UI (MySQL/MariaDB):

<Steps>
  <Step title="Mở Database Switcher">
    Click tên database trong toolbar hoặc `Cmd+K`
  </Step>
  <Step title="Click Create">
    Click **Create** trong toolbar của switcher
  </Step>
  <Step title="Cấu hình">
    Nhập tên, chọn character set và collation
  </Step>
  <Step title="Tạo">
    Click **Create** để thực thi
  </Step>
</Steps>

Character set và collation có sẵn:

| Charset | Collations |
|---------|------------|
| **utf8mb4** | utf8mb4_unicode_ci, utf8mb4_general_ci, utf8mb4_bin |
| **utf8** | utf8_unicode_ci, utf8_general_ci, utf8_bin |
| **latin1** | latin1_swedish_ci, latin1_general_ci, latin1_bin |
| **ascii** | ascii_general_ci, ascii_bin |

### Database Switcher

Chuyển giữa các database trên cùng kết nối:

1. Click tên database trong toolbar
2. Duyệt database có sẵn với metadata
3. Tìm theo tên để lọc
4. Nhấp đúp hoặc chọn rồi click **Open**

Switcher hiển thị database gần đây để truy cập nhanh và hỗ trợ refresh danh sách.

## Thao tác Collection MongoDB

Thao tác MongoDB khác SQL ở một số điểm.

### Tạo Collection

Tạo collection bằng MQL editor. Với tùy chọn cụ thể (capped, validator):

```javascript
db.createCollection("myCollection", {
  capped: true,
  size: 1048576,
  max: 1000
})
```

### Xóa Collection

Chuột phải collection trong sidebar, chọn **Drop Table**. Thực thi `db.collection.drop()`, xóa tất cả tài liệu và index.

### Hiển thị Tất cả Collections

Chọn **Show All Tables** từ sidebar để thực thi `listCollections` và hiển thị tất cả collection trong database hiện tại.

<Note>
**Truncate** không khả dụng cho MongoDB. Để xóa tất cả tài liệu mà giữ collection, dùng `db.collection.deleteMany({})` trong MQL editor.
</Note>
