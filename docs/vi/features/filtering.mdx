# Lọc Dữ Liệu

Filter panel xây dựng mệnh đề WHERE mà không cần viết SQL. Chọn cột, toán tử, giá trị trực quan, hoặc chuyển sang chế độ raw SQL cho những điều kiện phức tạp hơn. Filter hoạt động riêng biệt theo từng tab.

## Tổng Quan

Hệ thống lọc gồm năm phần:

- **Filter Panel** — trình tạo trực quan cho điều kiện WHERE
- **Quick Search** — tìm kiếm tức thì trên tất cả cột hiển thị
- **Filter Presets** — lưu và tải cấu hình filter có tên
- **SQL Preview** — xem mệnh đề WHERE được tạo trước khi áp dụng
- **Per-Tab State** — mỗi tab nhớ filter riêng

## Filter Panel

### Mở Filter Panel

Bật/tắt filter panel:

- Click nút **Filter** trong thanh công cụ
- Hoặc nhấn `Cmd+F`

Panel hiện phía trên data grid. Trường quick search tự động focus khi panel mở.

### Xây Dựng Filters

Mỗi hàng filter gồm ba phần:

1. **Column** — cột cần lọc
2. **Operator** — toán tử so sánh
3. **Value** — giá trị so sánh

### Thêm và Quản Lý Filters

| Thao tác | Cách thực hiện |
|----------|----------------|
| Thêm filter | Click nút **+** trong header |
| Sao chép filter | Click biểu tượng duplicate trên hàng filter |
| Xóa filter | Click nút **x** trên hàng filter |
| Focus vào filter | Click bất kỳ đâu trên hàng |

### Logic AND / OR

Chuyển đổi giữa AND và OR bằng segmented control trong header filter:

- **AND** — tất cả filter phải khớp (giao)
- **OR** — bất kỳ filter nào khớp (hợp)

<Tip>
AND thu hẹp kết quả dần dần. OR tìm hàng khớp với bất kỳ điều kiện nào.
</Tip>

### Chọn và Áp Dụng Filters

Mỗi hàng filter có checkbox để áp dụng có chọn lọc:

| Thao tác | Mô tả |
|----------|-------|
| **Chọn/bỏ chọn** | Bật/tắt checkbox trên từng filter |
| **Chọn tất cả** | Dùng checkbox "Select All" trong footer |
| **Áp dụng đơn** | Click nút play trên hàng filter để áp dụng riêng filter đó |
| **Áp dụng tất cả** | Click **Apply All** trong footer để áp dụng tất cả filter đã chọn |
| **Bỏ thiết lập** | Click **Unset** để xóa tất cả filter đã áp dụng và tải lại dữ liệu chưa lọc |

<Note>
Chỉ filter hợp lệ với đầy đủ cột, toán tử và giá trị (nếu cần) mới áp dụng được. Filter không hợp lệ hiển thị nút play bị vô hiệu.
</Note>

## Toán Tử

Filter builder hỗ trợ 18 toán tử:

| Toán tử | Tên Hiển Thị | Yêu Cầu Giá Trị | Mô tả |
|---------|--------------|-----------------|-------|
| `=` | equals | Có | Khớp chính xác |
| `!=` | not equals | Có | Loại trừ khớp chính xác |
| CONTAINS | contains | Có | Khớp chuỗi con (LIKE %value%) |
| NOT CONTAINS | not contains | Có | Loại trừ chuỗi con |
| STARTS WITH | starts with | Có | Khớp tiền tố (LIKE value%) |
| ENDS WITH | ends with | Có | Khớp hậu tố (LIKE %value) |
| `>` | greater than | Có | Lớn hơn |
| `>=` | greater or equal | Có | Lớn hơn hoặc bằng |
| `<` | less than | Có | Nhỏ hơn |
| `<=` | less or equal | Có | Nhỏ hơn hoặc bằng |
| IS NULL | is NULL | Không | Giá trị cột là NULL |
| IS NOT NULL | is not NULL | Không | Giá trị cột không phải NULL |
| IS EMPTY | is empty | Không | Giá trị cột là chuỗi rỗng |
| IS NOT EMPTY | is not empty | Không | Giá trị cột không phải chuỗi rỗng |
| IN | in list | Có | Giá trị trong danh sách phân tách bằng dấu phẩy |
| NOT IN | not in list | Có | Giá trị không trong danh sách |
| BETWEEN | between | Có (hai giá trị) | Giá trị nằm giữa hai giới hạn |
| REGEX | matches regex | Có | Khớp biểu thức chính quy |

### Toán Tử Không Cần Giá Trị

IS NULL, IS NOT NULL, IS EMPTY và IS NOT EMPTY ẩn trường giá trị và hiện dấu gạch ngang. Không cần nhập gì.

### Toán Tử BETWEEN

BETWEEN hiện hai trường giá trị với nhãn "and" ở giữa. Cả hai giá trị đều bắt buộc.

## Chế Độ Raw SQL

Khi visual builder không đáp ứng được, chuyển sang Raw SQL.

### Sử Dụng Raw SQL

1. Click dropdown cột trên hàng filter
2. Chọn **Raw SQL** ở đầu danh sách
3. Dropdown toán tử biến mất, trường giá trị chuyển thành input SQL tự do
4. Nhập điều kiện WHERE trực tiếp

{/* Screenshot: Raw SQL filter with arbitrary WHERE clause */}
<Frame caption="Chế độ filter Raw SQL">
  <img
    className="block dark:hidden"
    src="/images/filter-raw-sql.png"
    alt="Raw SQL filter"
  />
  <img
    className="hidden dark:block"
    src="/images/filter-raw-sql-dark.png"
    alt="Raw SQL filter"
  />
</Frame>

**Ví dụ:**

```sql
created_at > NOW() - INTERVAL 7 DAY
```

```sql
LOWER(name) LIKE '%smith%' AND status IN ('active', 'pending')
```

```sql
price * quantity > 1000
```

<Warning>
Raw SQL được chèn trực tiếp vào mệnh đề WHERE. Đảm bảo cú pháp đúng cho loại database đang kết nối.
</Warning>

|----------|----------------|
| Tải | Click tên preset trong menu folder |

<Note>

</Note>

## SQL Preview

Xem mệnh đề WHERE chính xác sẽ gửi đến database trước khi áp dụng.

### Xem Preview

1. Xây dựng filter
2. Click nút **SQL** trong footer filter
3. Một sheet hiển thị mệnh đề WHERE được tạo

### Preview Hiển Thị Gì

- Mệnh đề WHERE hoàn chỉnh với quoting phù hợp cho loại database
- MySQL dùng backticks (`` `column` ``)
- PostgreSQL dùng dấu nháy kép (`"column"`)
- Logic AND/OR theo chế độ hiện tại
- Nút "Copy" để sao chép SQL vào clipboard

Nếu một số (không phải tất cả) filter được chọn, preview chỉ hiển thị filter đã chọn. Ngược lại, hiển thị tất cả filter hợp lệ.

### Xử Lý Filters Không Hợp Lệ

Nếu có filter nhưng không filter nào hợp lệ, preview hiển thị thông báo chỉ ra cần hoàn thành gì:

- Chọn cột
- Nhập giá trị (nếu toán tử yêu cầu)
- Điền giá trị thứ hai cho BETWEEN

## Trạng Thái Filter theo Tab

Mỗi tab duy trì trạng thái filter riêng biệt.

### Cách Hoạt Động

- Mở bảng tạo trạng thái filter mới cho tab đó
- Chuyển tab giữ nguyên filter của mỗi tab
- Filter đã áp dụng vẫn hoạt động khi chuyển đi và quay lại
- Trạng thái hiển thị filter panel được nhớ theo tab

### Các Thành Phần Trạng Thái

Mỗi tab lưu:

| Thành phần | Mô tả |
|------------|-------|
| **Danh sách filter** | Tất cả hàng filter (đã áp dụng hay chưa) |
| **Filter đã áp dụng** | Filter đang hoạt động ảnh hưởng đến query |
| **Hiển thị panel** | Filter panel đang hiện hay ẩn |

### Cột Mặc Định

Chọn cột nào được chọn sẵn khi thêm filter mới:

| Tùy chọn | Mô tả |
|----------|-------|
| **Raw SQL** | Filter mới mặc định là chế độ Raw SQL |
| **Primary Key** | Filter mới mặc định là cột primary key |
| **Any Column** | Filter mới mặc định là cột có sẵn đầu tiên |

### Toán Tử Mặc Định

Chọn toán tử nào được chọn sẵn khi thêm filter mới:

| Tùy chọn | Mô tả |
|----------|-------|
| **Equal (=)** | Mặc định khớp chính xác |
| **Contains** | Mặc định khớp chuỗi con |

### Trạng Thái Panel khi Mở

Hành vi filter panel khi mở bảng:

| Tùy chọn | Mô tả |
|----------|-------|
| **Always Hide** | Filter panel bắt đầu ẩn (mặc định) |
| **Always Show** | Filter panel luôn hiện khi mở bảng |
| **Restore Last Filter** | Khôi phục filter cuối cùng dùng trên bảng đó |

<Tip>
"Restore Last Filter" nhớ filter theo tên bảng. Nếu bạn hay lọc cùng bảng với cùng điều kiện, tùy chọn này tiết kiệm thời gian thiết lập mỗi phiên.
</Tip>
