# Build TablePro

## Build Phát Triển

### Dùng Xcode

1. Mở `TablePro.xcodeproj`
2. Chọn scheme **TablePro**
3. Chọn **My Mac** làm destination
4. `Cmd+R` để build và chạy, hoặc `Cmd+B` chỉ build

### Dùng Command Line

```bash
# Build kiến trúc hiện tại (Debug)
xcodebuild -project TablePro.xcodeproj \
    -scheme TablePro \
    -configuration Debug \
    build \
    -skipPackagePluginValidation

# Build và chạy
xcodebuild -project TablePro.xcodeproj \
    -scheme TablePro \
    -configuration Debug \
    build \
    -skipPackagePluginValidation && open build/Debug/TablePro.app
```

### Build Sạch

Nếu gặp lỗi build:

```bash
# Clean qua xcodebuild
xcodebuild -project TablePro.xcodeproj -scheme TablePro clean

# Hoặc clean derived data
rm -rf ~/Library/Developer/Xcode/DerivedData

# Hoặc trong Xcode: Cmd+Shift+K
```

## Build Release

### Dùng Build Script

```bash
# Apple Silicon (M1/M2/M3/M4/M5)
scripts/build-release.sh arm64

# Intel
scripts/build-release.sh x86_64

# Cả hai kiến trúc
scripts/build-release.sh both
```

### Chi tiết Build Script

`build-release.sh` thực hiện:

1. **Chuẩn bị thư viện**: Trích kiến trúc đúng từ thư viện universal
2. **Build app**: Chạy xcodebuild cấu hình Release
3. **Sao chép output**: Đặt app trong `build/Release/`
4. **Khôi phục icon**: Đảm bảo app icon đầy đủ
5. **Xác minh kiến trúc**: Xác nhận binary khớp target

### Vị trí Output

```
build/Release/TablePro-arm64.app   # Apple Silicon
build/Release/TablePro-x86_64.app  # Intel
```

### Build Release Thủ Công

```bash
# Đặt kiến trúc
ARCH=arm64  # hoặc x86_64

# Build
xcodebuild \
    -project TablePro.xcodeproj \
    -scheme TablePro \
    -configuration Release \
    -arch $ARCH \
    ONLY_ACTIVE_ARCH=YES \
    CODE_SIGN_IDENTITY="" \
    CODE_SIGNING_REQUIRED=NO \
    clean build
```

## Thư Viện Native

TablePro dùng thư viện C native cho kết nối database.

### Tệp thư viện

Trong `Libs/`:

| File | Mục đích |
|------|---------|
| `libmariadb_universal.a` | MariaDB universal (arm64 + x86_64) |
| `libmariadb.a` | Đặc thù kiến trúc (tạo khi build) |

### Thiết lập thư viện

Thư viện universal phải tồn tại trước khi build:

```bash
# Kiểm tra
ls -la Libs/libmariadb_universal.a

# Nếu thiếu, tạo từ thư viện từng kiến trúc
lipo -create \
    Libs/libmariadb_arm64.a \
    Libs/libmariadb_x86_64.a \
    -output Libs/libmariadb_universal.a
```

Build script tự động trích slice kiến trúc đúng.

## Tạo DMG

### Dùng DMG Script

```bash
# Build release trước
scripts/build-release.sh arm64

# Rồi tạo DMG
scripts/create-dmg.sh arm64
```

### Nội dung DMG

- TablePro.app
- Shortcut thư mục Applications

- License file (nếu có)

```

build/Release/TablePro-x86_64.dmg
```

## Ký Code

### Build Phát Triển

Không ký mặc định. Đủ dùng cho test cục bộ.

### Phân Phối Release

<Warning>
TablePro hiện phân phối không có chữ ký. macOS Gatekeeper hiện cảnh báo khi chạy lần đầu.
</Warning>

Để ký (cần tài khoản Apple Developer):

```bash
# Ký app
codesign --force --deep --sign "Developer ID Application: Your Name (TEAM_ID)" \
    build/Release/TablePro-arm64.app

# Xác minh chữ ký
codesign --verify --verbose build/Release/TablePro-arm64.app
```

### Notarization

Cho phân phối ngoài App Store:

```bash
# Tạo ZIP
ditto -c -k --keepParent build/Release/TablePro-arm64.app TablePro.zip

# Gửi notarization
xcrun notarytool submit TablePro.zip \
    --apple-id "your@email.com" \
    --team-id "TEAM_ID" \
    --password "app-specific-password" \
    --wait

# Staple ticket
xcrun stapler staple build/Release/TablePro-arm64.app
```

## Build Configuration

### Cài Đặt Build

| Setting | Debug | Release |
|---------|-------|---------|
| Optimization | None (-O0) | Optimize for Speed (-O) |
| Debug Information | Full | Stripped |
| Assertions | Enabled | Disabled |
| Code Signing | None | Developer ID (optional) |

### Cài Đặt Kiến Trúc

| Setting | Value |
|---------|-------|
| ARCHS | Standard (arm64, x86_64) |
| ONLY_ACTIVE_ARCH | Yes (Debug), No (Release) |
| VALID_ARCHS | arm64 x86_64 |

## Khắc Phục Sự Cố

### Lỗi Build

**Thiếu thư viện**:
```
ld: library not found for -lmariadb
```

Sửa:
1. Kiểm tra thư mục `Libs/`
2. Chạy `scripts/build-release.sh` để chuẩn bị thư viện
3. Xác minh `libmariadb.a` tồn tại cho kiến trúc target

**Lỗi code signing**:
```
Code signing is required for product type 'Application'
```

Sửa: Build script tắt code signing. Build thủ công thì thêm:
```
CODE_SIGN_IDENTITY=""
CODE_SIGNING_REQUIRED=NO
```

**Không khớp kiến trúc**:
```
building for macOS-arm64 but attempting to link with file built for macOS-x86_64
```

Sửa:
1. Clean build folder
2. Đảm bảo thư viện khớp kiến trúc target
3. Dùng build script xử lý kiến trúc thư viện

### Vấn đề Xcode

**Không tìm thấy scheme**:
1. Mở Xcode
2. Product > Scheme > Manage Schemes
3. Đảm bảo TablePro scheme được check

**Derived data**:
```bash
rm -rf ~/Library/Developer/Xcode/DerivedData
```

**Index**:
```bash
rm -rf ~/Library/Developer/Xcode/DerivedData/*/Index
```

## CI/CD

### GitHub Actions

Workflow CI mẫu:

```yaml
name: Build

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  build:
    runs-on: macos-latest
    steps:
      - uses: actions/checkout@v4

      - name: Select Xcode
        run: sudo xcode-select -s /Applications/Xcode_15.0.app

      - name: Build
        run: |
          xcodebuild -project TablePro.xcodeproj \
            -scheme TablePro \
            -configuration Debug \
            build \
            -skipPackagePluginValidation

      - name: Test
        run: |
          xcodebuild -project TablePro.xcodeproj \
            -scheme TablePro \
            test \
            -skipPackagePluginValidation
```

### Quy Trình Release

```yaml
release:
  runs-on: macos-latest
  steps:
    - uses: actions/checkout@v4

    - name: Build Release (arm64)
      run: scripts/build-release.sh arm64

    - name: Build Release (x86_64)
      run: scripts/build-release.sh x86_64

    - name: Create DMGs
      run: |
        scripts/create-dmg.sh arm64
        scripts/create-dmg.sh x86_64

    - name: Upload Artifacts
      uses: actions/upload-artifact@v3
      with:
        name: release-dmgs
        path: build/Release/*.dmg
```
