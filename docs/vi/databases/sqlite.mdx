# Cơ sở dữ liệu SQLite

SQLite là database engine độc lập, dạng file. Không cần server. Toàn bộ database nằm trong một file duy nhất trên Mac.

## Thiết lập nhanh

<Steps>
  <Step title="Mở Form kết nối">
    Nhấp **New Connection** từ màn hình Chào mừng hoặc **File** > **New Connection**
  </Step>
  <Step title="Chọn SQLite">
    Chọn **SQLite** từ danh sách loại database
  </Step>
  <Step title="Chọn file Database">
    Nhấp **Browse** và chọn file `.sqlite`, `.db` hoặc `.sqlite3`
  </Step>
  <Step title="Tạo kết nối">
    Đặt tên kết nối và nhấp **Create**
  </Step>
</Steps>

## Vị trí SQLite phổ biến

### Database ứng dụng

Nhiều ứng dụng macOS dùng SQLite:

| Ứng dụng | Vị trí Database |
|-------------|-------------------|
| Safari | `~/Library/Safari/History.db` |
| Photos | `~/Pictures/Photos Library.photoslibrary/database/Photos.sqlite` |
| Messages | `~/Library/Messages/chat.db` |
| Notes | `~/Library/Group Containers/group.com.apple.notes/NoteStore.sqlite` |

<Warning>
Database hệ thống có thể bị khóa khi ứng dụng mẹ đang chạy. Thoát ứng dụng trước khi mở database.
</Warning>

### Database phát triển

Vị trí phổ biến:

```
# Ruby on Rails
./db/development.sqlite3

# Django
./db.sqlite3

# iOS Simulator
~/Library/Developer/CoreSimulator/Devices/*/data/Containers/Data/Application/*/Documents/*.sqlite

# Core Data
*.sqlite (trong container app)
```

### Tạo Database mới

Tạo database SQLite mới:

1. Tạo kết nối mới
2. Nhập đường dẫn file chưa tồn tại
3. TablePro tạo file khi bạn kết nối

Hoặc từ command line:

```bash
sqlite3 ~/Documents/newdatabase.sqlite "SELECT 1;"
```

### Thực thi truy vấn

Hỗ trợ đầy đủ cú pháp SQLite:

```sql
-- JSON functions (SQLite 3.38+)
SELECT json_extract(data, '$.name') as name
FROM users
WHERE json_extract(data, '$.active') = true;

-- Window functions
SELECT
    category,
    product,
    price,
    rank() OVER (PARTITION BY category ORDER BY price DESC) as rank
FROM products;

-- CTEs
WITH RECURSIVE
    cnt(x) AS (
        SELECT 1
        UNION ALL
        SELECT x+1 FROM cnt WHERE x < 10
    )
SELECT x FROM cnt;

-- Full-text search (nếu FTS được bật)
SELECT * FROM documents WHERE documents MATCH 'sqlite AND database';

-- UPSERT (INSERT OR REPLACE)
INSERT INTO settings (key, value)
VALUES ('theme', 'dark')
ON CONFLICT(key) DO UPDATE SET value = excluded.value;
```

## Kiểu dữ liệu SQLite

SQLite dùng hệ thống kiểu động với type affinity:

| Affinity | Mô tả | Ví dụ |
|----------|-------------|----------|
| TEXT | Dữ liệu chuỗi | `VARCHAR`, `TEXT`, `CLOB` |
| INTEGER | Signed integer | `INT`, `INTEGER`, `BIGINT` |
| REAL | Floating point | `REAL`, `DOUBLE`, `FLOAT` |
| NUMERIC | Numeric hoặc text | `DECIMAL`, `BOOLEAN`, `DATE` |
| BLOB | Binary data | `BLOB`, raw bytes |

<Note>
SQLite chấp nhận bất kỳ kiểu nào trong bất kỳ cột nào (trừ INTEGER PRIMARY KEY). Kiểu là gợi ý, không phải constraint.
</Note>

## Làm việc với file lớn

### Kích thước file

Database SQLite có thể lớn. TablePro xử lý:

- Database đến vài GB
- Bảng hàng triệu dòng
- Dữ liệu BLOB (hình ảnh, files)

### Mẹo hiệu năng

1. **Bật phân trang**: Trong settings cho bảng lớn
2. **Thêm indexes**: Cho cột query thường xuyên
3. **Dùng LIMIT**: Luôn giới hạn khi khám phá
4. **Vacuum định kỳ**: Thu hồi không gian với `VACUUM`

```sql
-- Kiểm tra kích thước database
SELECT page_count * page_size as size FROM pragma_page_count(), pragma_page_size();

-- Phân tích bảng
ANALYZE;

-- Thu hồi không gian
VACUUM;
```

## Khắc phục sự cố

### Database is Locked

**Triệu chứng**: "database is locked" hoặc "database table is locked"

**Nguyên nhân**:
- Ứng dụng khác đang mở database
- Truy vấn trước vẫn chạy
- Xung đột WAL mode

**Giải pháp**:

1. Đóng ứng dụng khác đang dùng database
2. Chờ truy vấn hoàn thành
3. Kiểm tra file WAL:
   ```bash
   ls -la /path/to/database.sqlite*
   # Có thể thấy: database.sqlite-wal, database.sqlite-shm
   ```

### Cannot Open Database

**Triệu chứng**: "unable to open database file"

**Nguyên nhân và Giải pháp**:

1. **File không tồn tại**: Kiểm tra đường dẫn
2. **Permission denied**: Kiểm tra quyền file
   ```bash
   ls -la /path/to/database.sqlite
   chmod 644 /path/to/database.sqlite
   ```
3. **Đường dẫn chứa ký tự đặc biệt**: Dùng đường dẫn tuyệt đối

### Corrupt Database

**Triệu chứng**: "database disk image is malformed"

**Cách khôi phục**:

1. **Kiểm tra toàn vẹn**:
   ```sql
   PRAGMA integrity_check;
   ```

2. **Thử khôi phục**:
   ```bash
   sqlite3 corrupt.sqlite ".recover" | sqlite3 recovered.sqlite
   ```

3. **Khôi phục từ backup** (nếu có)

### Thay đổi Schema không hiển thị

**Triệu chứng**: Bảng hoặc cột tạo mới không hiện

**Giải pháp**: Làm mới kết nối:
1. Chuột phải kết nối
2. Chọn **Refresh**

Hoặc ngắt kết nối rồi kết nối lại.

## SQLite Pragmas

TablePro hỗ trợ lệnh PRAGMA SQLite:

```sql
-- Kiểm tra phiên bản SQLite
SELECT sqlite_version();

-- Thông tin bảng
PRAGMA table_info(users);

-- Kiểm tra foreign keys
PRAGMA foreign_keys;

-- Bật foreign key enforcement
PRAGMA foreign_keys = ON;

-- Kiểm tra journal mode
PRAGMA journal_mode;

-- Dùng WAL mode cho đồng thời tốt hơn
PRAGMA journal_mode = WAL;
```

## Backup và Restore

### Backup thủ công

Database SQLite chỉ là file. Backup:

```bash
# Copy đơn giản (đảm bảo database không đang ghi)
cp database.sqlite database-backup.sqlite

# Lệnh backup SQLite (an toàn hơn, xử lý locks)
sqlite3 database.sqlite ".backup backup.sqlite"
```

### Export sang SQL

```bash
sqlite3 database.sqlite .dump > backup.sql
```

### Restore từ SQL

```bash
sqlite3 newdatabase.sqlite < backup.sql
```

## Khác biệt với Server Databases

| Tính năng | SQLite | MySQL/PostgreSQL |
|---------|--------|------------------|
| Cần server | Không | Có |
| Nhiều kết nối | Giới hạn | Không giới hạn |
| Xác thực | Không | Có |
| Truy cập mạng | Dạng file | TCP/IP |
| Ghi đồng thời | Giới hạn | Đầy đủ |
| Stored procedures | Không | Có |
| Kiểu dữ liệu | Dynamic | Static |

## Trường hợp sử dụng

SQLite phù hợp cho:

- **Phát triển local**: Prototype ứng dụng nhanh
- **Embedded apps**: Mobile, desktop apps
- **Single-user apps**: Database cá nhân
- **Phân tích dữ liệu**: Khám phá dữ liệu nhanh
- **Testing**: Unit tests với isolated databases
- **Prototyping**: Trước khi migrate sang server database
