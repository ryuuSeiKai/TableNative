# Cài đặt

## Yêu cầu Hệ thống

### Yêu cầu Tối thiểu

| Yêu cầu | Thông số kỹ thuật |
|-------------|---------------|
| **Phiên bản macOS** | macOS 14.0 (Sonoma) hoặc mới hơn |
| **Bộ xử lý** | Apple Silicon (M1/M2/M3/M4/M5) hoặc Intel x86_64 |
| **Bộ nhớ** | Tối thiểu 4 GB RAM |
| **Dung lượng** | ~200 MB dung lượng trống |

### Khuyến nghị

- **macOS**: Phiên bản mới nhất
- **Bộ nhớ**: 8 GB RAM trở lên cho database lớn
- **Dung lượng**: SSD để query history và tab persistence nhanh hơn

<Note>
TablePro dùng framework gốc Apple. Không cần Java, .NET hay runtime nào khác.
</Note>

## Cài đặt qua Homebrew

Cách nhanh nhất:

```bash
brew install --cask tablepro
```

<Tip>
Công thức cask tự động xóa thuộc tính quarantine, nên không cần chạy `xattr -cr` khi mở lần đầu.
</Tip>

Cập nhật:

```bash
brew upgrade tablepro
```

Gỡ cài đặt:

```bash
brew uninstall tablepro
```

Xóa cả dữ liệu ứng dụng:

```bash
brew zap tablepro
```

## Tải xuống

### Từ GitHub Releases

1. Truy cập trang [TablePro Releases](https://github.com/datlechin/tablepro/releases)
2. Tìm bản phát hành mới nhất
3. Tải DMG phù hợp:

| Kiến trúc | File | Dành cho Mac với |
|--------------|------|---------------|
| Apple Silicon | `TablePro-arm64.dmg` | Chip M1, M2, M3, M4, M5 |
| Intel | `TablePro-x86_64.dmg` | Bộ xử lý Intel |

### Kiểm tra Kiến trúc Mac

1. Nhấp **menu Apple** () trên thanh menu
2. Chọn **About This Mac**
3. Tìm:
   - **Chip**: Apple M1/M2/M3/M4/M5 = Apple Silicon
   - **Processor**: Intel = x86_64

## Các bước Cài đặt

### Cài đặt Tiêu chuẩn

1. **Mở file DMG**
   - Nhấp đúp file `.dmg` đã tải
   - Cửa sổ Finder mới hiện ra

2. **Kéo vào Applications**
   - Kéo icon TablePro vào shortcut thư mục Applications
   - Đợi copy xong

4. **Khởi chạy TablePro**
   - Mở **Finder** > **Applications**
   - Nhấp đúp **TablePro**

### Bảo mật Khởi chạy Lần đầu

TablePro chưa được ký chứng chỉ Apple Developer. macOS sẽ báo lỗi **"TablePro is damaged and can't be opened"** khi mở lần đầu. Đây là hành vi quarantine tiêu chuẩn với app unsigned.

<Warning>
Bình thường với app unsigned tải từ internet. TablePro là source-available, mã nguồn công khai để ai cũng kiểm tra được.
</Warning>

#### Xóa thuộc tính quarantine

Mở **Terminal** và chạy:

```bash
xattr -cr /Applications/TablePro.app
```

Sau đó mở TablePro bình thường từ Applications.

<Tip>
Chỉ cần làm một lần. Sau khi xóa quarantine, TablePro mở bình thường mọi lần sau.
</Tip>

#### Cách khác: System Settings

Nếu vẫn thấy cảnh báo bảo mật sau khi chạy lệnh:

1. Mở **System Settings** > **Privacy & Security**
2. Cuộn xuống phần **Security**
3. Sẽ thấy thông báo TablePro bị chặn
4. Nhấp **Open Anyway**
5. Nhập mật khẩu nếu được yêu cầu

## Cập nhật TablePro

### Cập nhật Tự động

TablePro kiểm tra bản mới trong nền qua [Sparkle](https://sparkle-project.org/) và thông báo khi có cập nhật.

### Kiểm tra Cập nhật Thủ công

Kiểm tra thủ công:

1. Mở menu **TablePro** trên thanh menu
2. Nhấp **Check for Updates...**

### Cài đặt Cập nhật

Cấu hình trong **Settings** > **General** > **Software Update**:

- Bật/tắt kiểm tra cập nhật tự động
- Chọn tần suất kiểm tra

<Tip>
Kết nối và cài đặt lưu riêng biệt với app, không bị mất khi cập nhật.
</Tip>

### Vị trí Lưu trữ Dữ liệu

| Dữ liệu | Vị trí | Loại lưu trữ |
|------|----------|--------------|
| **Kết nối** | `~/Library/Preferences/com.TablePro.plist` | UserDefaults |
| **Cài đặt** | `~/Library/Preferences/com.TablePro.plist` | UserDefaults |
| **Mật khẩu** | macOS Keychain (lưu trữ an toàn) | Keychain |
| **Lịch sử truy vấn** | `~/Library/Application Support/TablePro/query_history.db` | SQLite |

## Gỡ cài đặt

### Xóa Ứng dụng

1. Thoát TablePro nếu đang chạy
2. Mở **Finder** > **Applications**
3. Kéo **TablePro** vào Thùng rác

### Xóa Dữ liệu Ứng dụng (Tùy chọn)

Xóa toàn bộ dữ liệu TablePro:

```bash
# Xóa dữ liệu application support
rm -rf ~/Library/Application\ Support/TablePro

# Xóa preferences
rm ~/Library/Preferences/com.TablePro.plist

# Xóa Keychain items (qua app Keychain Access)
# Tìm "TablePro" và xóa các mục liên quan
```

<Warning>
Thao tác này xóa toàn bộ kết nối đã lưu và lịch sử truy vấn. Xuất dữ liệu quan trọng trước.
</Warning>

## Khắc phục Sự cố

### "TablePro is damaged and can't be opened"

Làm theo hướng dẫn [Bảo mật Khởi chạy Lần đầu](#bảo-mật-khởi-chạy-lần-đầu) để xóa thuộc tính quarantine.

### "TablePro can't be opened because it is from an unidentified developer"

Làm theo hướng dẫn [Bảo mật Khởi chạy Lần đầu](#bảo-mật-khởi-chạy-lần-đầu).

### Ứng dụng crash khi khởi chạy

1. Kiểm tra đã tải đúng phiên bản kiến trúc chưa
2. Đảm bảo đang chạy macOS 14.0 trở lên
3. Thử xóa và cài lại
4. Kiểm tra Console.app để xem crash logs

### Vấn đề kết nối sau khi cài đặt

Nếu TablePro mở được nhưng không kết nối được database:

1. Kiểm tra database server đang chạy
2. Kiểm tra firewall (System Settings > Network > Firewall)
3. Đảm bảo database server cho phép kết nối từ localhost

## Build từ Source

Dành cho developer muốn build TablePro từ source:

1. Clone repository:
```bash
git clone https://github.com/datlechin/tablepro.git
cd tablepro
```

2. Mở project Xcode:
```bash
open TablePro.xcodeproj
```

3. Build và chạy:
   - Chọn scheme **TablePro**
   - Nhấn `Cmd+R` để build và chạy

Xem hướng dẫn [Development Setup](/vi/development/setup) chi tiết hơn.
