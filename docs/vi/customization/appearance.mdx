## Chủ đề

Ba chế độ:

| Chủ đề | Mô tả |
|-------|-------------|
| **System** | Theo giao diện macOS tự động (mặc định) |
| **Light** | Luôn sáng |
| **Dark** | Luôn tối |

### Chủ đề Hệ thống

Khi đặt **System**, TablePro theo giao diện macOS:

1. Mở **System Settings** > **Appearance**
2. Chọn Light, Dark hoặc Auto
3. TablePro tự động cập nhật

### Chủ đề Sáng

- Nền trắng/xám nhạt
- Văn bản tối dễ đọc
- Phù hợp môi trường đủ sáng
- Độ tương phản thấp cho phòng sáng

### Chủ đề Tối

- Nền xám đậm/đen
- Văn bản sáng tạo tương phản
- Giảm mỏi mắt trong ánh sáng yếu
- Tiết kiệm pin trên màn hình OLED

<Tip>
Chủ đề tối phù hợp cho phiên làm việc dài, đặc biệt trong môi trường thiếu sáng.
</Tip>

### Khớp với macOS

Để dùng màu nhấn hệ thống:

1. Đặt màu nhấn TablePro thành **System**
2. Mở **System Settings** > **Appearance**
3. Chọn màu ưa thích
4. TablePro tự động cập nhật

## Màu Kết nối

Gán màu cho kết nối để tổ chức trực quan:

| Màu | Đề xuất |
|-------|---------------|
| None | Mặc định, trung tính |
| Red | Database Production |
| Orange | Môi trường Staging |
| Yellow | Database Testing |
| Green | Development/local |
| Blue | Database Shared |
| Purple | Mục đích đặc biệt |
| Pink | Dự án cá nhân |

### Đặt Màu Kết nối

1. Mở form kết nối (chỉnh sửa hoặc tạo mới)
2. Trong phần **Appearance**, nhấp **Color**
3. Chọn màu từ bảng màu
4. Lưu kết nối

## Chủ đề Editor

Màu tô sáng cú pháp SQL:

| Phần tử | Màu |
|---------|-------|
| Keywords | Hồng |
| Strings | Xanh lá |
| Numbers | Xanh dương |
| Comments | Xám |
| Identifiers | Màu văn bản mặc định |

Tự động thích ứng với chủ đề sáng/tối.

## Phần tử UI

### Màu Trạng thái

| Màu | Ý nghĩa |
|-------|---------|
| Green | Thành công, đã kết nối, hợp lệ |
| Orange | Cảnh báo, đang xử lý, đang kết nối |
| Red | Lỗi, thất bại, không hợp lệ |
| Blue | Thông tin |
| Gray | Trung tính, đã ngắt kết nối |

### Màu Kiểu Dữ liệu

Trong bảng dữ liệu:

| Loại | Hiển thị |
|------|---------|
| NULL | Xám, in nghiêng |
| Boolean True | Xanh lá |
| Boolean False | Đỏ |
| Numbers | Màu tiêu chuẩn |
| Text | Màu tiêu chuẩn |

## Thực hành Tốt

### Cho Khả năng Đọc

1. **Khớp môi trường**: Tối trong ánh sáng yếu, sáng trong phòng sáng
2. **Độ tương phản**: Đảm bảo đủ tương phản cho điều kiện ánh sáng
3. **Kích thước font**: Tăng nếu khó đọc

### Cho Tổ chức

1. **Màu kết nối**: Chọn bảng màu nhất quán
2. **Mã hóa môi trường**: Đỏ=prod, xanh lá=dev, v.v.
3. **Quy ước nhóm**: Thống nhất ý nghĩa màu

### Cho Sức khỏe Mắt

1. **Chủ đề tối vào đêm**: Giảm ánh sáng xanh
2. **Nghỉ giải lao**: Nhìn xa màn hình định kỳ
3. **Night Shift**: Bật Night Shift macOS cho màu ấm hơn

## Khả năng Tiếp cận

### Mù Màu

1. Không chỉ dựa vào màu để truyền đạt ý nghĩa
2. Tên kết nối cung cấp nhận dạng bằng văn bản
3. Biểu tượng bổ sung thông tin màu

### Độ Tương phản Cao

1. Dùng chủ đề sáng hoặc tối (không auto)
2. Chọn màu nhấn tương phản cao (xanh dương, xanh lá)
3. Tăng kích thước font trong cài đặt editor

## Khắc phục Sự cố

### Chủ đề Không Đổi

1. Khởi động lại TablePro
2. Kiểm tra giao diện macOS nếu dùng System
3. Xác minh cài đặt đã lưu

### Màu Trông Sai

1. Kiểm tra color profile trong System Settings
2. Xác minh hiệu chỉnh màn hình
3. Kiểm tra True Tone trên màn hình hỗ trợ

### Màu nhấn Không Áp dụng

1. Khởi động lại TablePro
2. Đặt màu cụ thể thay vì System
3. Kiểm tra extension hệ thống xung đột
