## Cài đặt Font

### Họ Font

Các font monospace có sẵn:

| Font | Mô tả |
|------|-------------|
| **System Mono** | Monospace mặc định macOS (khuyến nghị) |
| **SF Mono** | San Francisco Mono |
| **Menlo** | Monospace cổ điển macOS |
| **Monaco** | Font lập trình truyền thống Mac |
| **Courier New** | Chuẩn đa nền tảng |

<Tip>
**System Mono** tự động dùng font monospace hệ thống tốt nhất.
</Tip>

### Tính khả dụng Font

| Font | Khả dụng |
|------|--------------|
| System Mono | Luôn có |
| SF Mono | macOS 10.12+ |
| Menlo | macOS 10.6+ |
| Monaco | macOS Classic |
| Courier New | Font hệ thống chuẩn |

Nếu font chọn không khả dụng, TablePro fallback về System Mono.

## Cài đặt Hiển thị

### Số Dòng

| Tùy chọn | Mô tả |
|--------|-------------|
| **On** | Hiện số dòng (mặc định) |
| **Off** | Ẩn cho giao diện tối giản |

Số dòng hữu ích khi:

- Tham chiếu dòng cụ thể
- Error message đề cập số dòng
- Điều hướng query lớn
- Thảo luận code với người khác

### Xuống Dòng

| Tùy chọn | Mô tả |
|--------|-------------|
| **Off** | Dòng dài cuộn ngang (mặc định) |
| **On** | Dòng dài tự động xuống dòng |

**Bật xuống dòng khi**:

- Làm việc trên màn hình hẹp
- Query có dòng dài
- Muốn tránh cuộn ngang

**Tắt xuống dòng khi**:

- Muốn độ dài dòng nhất quán
- Làm việc với query đã format
- Cần thấy rõ cấu trúc query

## Cài đặt Thụt lề

### Độ rộng Tab

| Cài đặt | Tùy chọn | Mặc định |
|---------|---------|---------|
| Độ rộng Tab | 1-16 khoảng trắng | 4 |

Lựa chọn phổ biến:

| Độ rộng | Sử dụng |
|-------|-------|
| 2 khoảng trắng | Gọn, kiểu JavaScript |
| 4 khoảng trắng | SQL chuẩn (khuyến nghị) |
| 8 khoảng trắng | Truyền thống, kiểu Unix |

## Hành vi Editor

### Nhập Văn bản

Editor hoạt động như trình soạn thảo macOS chuẩn:

- Phím tắt chuẩn hoạt động
- Copy/paste bình thường
- Lịch sử undo/redo được duy trì
- Chọn vùng bằng chuột và bàn phím

### Tự động hoàn thành

Autocomplete luôn bật, dismiss bằng `Escape`.

Xem [Tự động hoàn thành](/vi/features/autocomplete) để biết chi tiết.

## Tô sáng Cú pháp

SQL tự động tô sáng:

| Phần tử | Màu (Tối) | Màu (Sáng) |
|---------|--------------|---------------|
| Keywords | Hồng | Hồng |
| Strings | Xanh lá | Xanh lá |
| Numbers | Xanh dương | Xanh dương |
| Comments | Xám | Xám |
| Functions | Mặc định | Mặc định |
| Identifiers | Trắng | Đen |

Màu thích ứng theo chủ đề hiện tại.

### Keywords được Tô sáng

- `SELECT`, `INSERT`, `UPDATE`, `DELETE`, `CREATE`, `ALTER`, `DROP`
- `FROM`, `WHERE`, `JOIN`, `ON`, `GROUP BY`, `ORDER BY`, `HAVING`
- `AND`, `OR`, `NOT`, `IN`, `LIKE`, `BETWEEN`, `IS NULL`
- `INNER`, `LEFT`, `RIGHT`, `OUTER`, `CROSS`, `UNION`
- Và nhiều keywords SQL chuẩn khác

## Phím tắt trong Editor

| Hành động | Phím tắt |
|--------|----------|
| Thực thi query | `Cmd+Enter` |
| Hoàn tác | `Cmd+Z` |
| Làm lại | `Cmd+Shift+Z` |
| Tìm | `Cmd+F` |
| Thay thế | `Cmd+Option+F` |
| Chọn tất cả | `Cmd+A` |
| Đi đến dòng | `Cmd+G` |

Xem [Phím tắt](/vi/features/keyboard-shortcuts) cho danh sách đầy đủ.

## Khắc phục Sự cố

### Font Không Hiển thị Đúng

1. Xác minh font đã cài trên hệ thống
2. Thử font khác
3. Khởi động lại TablePro

### Cài đặt Không Lưu

1. Kiểm tra quyền đĩa
2. Xác minh file preferences ghi được:
   ```bash
   ls -la ~/Library/Preferences/com.TablePro.plist
   ```
3. Thử đặt lại preferences

### Editor Chậm

1. Giảm kích thước font (font lớn render chậm hơn)
2. Tắt xuống dòng cho query dài
3. Kiểm tra query quá lớn trong editor
